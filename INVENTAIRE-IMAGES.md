## Spécifications techniques

### Photos (WebP)
- **Hero** : 1920×1080px minimum, 16:9
- **Cards/Vignettes** : 800×600px, 4:3
- **Portraits** : 600×800px, 3:4
- **Pièces musée** : 800×800px, 1:1
- **Qualité** : 80-85% WebP
- **Poids max** : 200Ko pour cards, 500Ko pour hero

### Logos (SVG)
- Format vectoriel SVG privilégié
- Fallback PNG si SVG indisponible : 320×160px @2x
- Fond transparent
